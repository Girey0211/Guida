# Changelog

## [0.3.5] - 2026-06-16
### Added
- 앱 icon 및 favicon 통일

## [0.3.4] - 2026-06-16

### Added

- Sybil IP 기반 좋아요 락 기능 추가
- Revocation/이관 API 추가

### Fixed

- Revocation 추가에 따른 대규모 구조 변경

## [0.3.3] - 2026-06-16

### Added

- 패치노트 추가 및 CI에 패치노트 포함
- 루트 삭제 기능 추가

### Fixed

- 랜딩 페이지에서 다운로드 버튼을 누를 시 릴리즈 탭으로 이동이 아닌 파일이 바로 다운로드 되도록 수정
- 내 프로필 페이지를 개별 페이지가 아닌 탭 형식으로 변경 및 다른 유저의 프로필은 창으로 표시되도록 수정
- 보안 관련 로직 개선 및 보안 사항에 걸리는 요청 양식 수정
- 업데이트 방식 윈도우 패시브 모드로 변경

### Changed

- 임시 변경

### Removed

- 임시 삭제

### Deprecated

- 임시 비권장

### Security

- 임시 보안 개선

## [0.3.2] - 2026-06-16

### Added

- E.G.O 기프트 멀티 조건 조합 지원 (우미다 및 저택의 부산물 등 개별 획득 제한 적용)
- 기프트 픽커(추가) 페이지 및 플레이 스크린에 독립형 E.G.O 기프트 필터 기능 추가
- 기프트 픽커 페이지에 기프트 태그, 조합식 정보, 선행 기프트 정보 및 루트 추가 여부 표시 기능 추가

### Fixed

- 플레이 스크린 E.G.O 기프트 UI에서 타겟 층 및 난이도 태그 제거
- 획득 여부 상태 색상을 조합식 보기와 통합하고 선행 요구사항 알림 스타일 통일
- 일반 조합 기프트가 선행 기프트를 모두 획득하기 전까지 잠금 상태가 유지되도록 수정
- 달의 기억(Memory of Lunar) 기프트 획득 조건 정렬 로직 보완

## [0.3.1] - 2026-06-15

### Added

- 사용자 프로필 관리 기능 추가 (랜덤 UUID 닉네임 생성, 설명 편집)
- 루트 탐색 탭에서 본인이 공유한 루트를 포함하여 검색할 수 있도록 필터 수정
- 탐색 완료 시 루트 플레이 카운트가 증가하도록 로직 연동 및 마이 루트 탭에서 조회 지원
- 가져온(Import) 루트에 대한 정합성 검증 상태 동기화 문제 수정

### Fixed

- 윈도우 시작 시 global-shortcut 플러그인 F9 키 등록 실패로 인한 크래시 해결
- 공유/가져온 루트에 대해 수정 권한이 없더라도 읽기 전용 상세 보기를 지원하는 버튼 추가
- 탐색 종료 팝업의 디자인을 앱 구동 공지 팝업과 통일
